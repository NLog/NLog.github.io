NOTE! Obsolete, instead override WriteAsyncThreadSafe(IList{AsyncLogEventInfo} logEvents) Writes an array of logging events to the log target, in a thread safe manner. !WARNING! Custom targets should only override this method if able to provide their own synchronization mechanism. Layout-objects are not guaranteed to be threadsafe, so using them without a SyncRoot-object can be dangerous.
